## Project Overview

**Quiz Master** is a comprehensive quiz management system developed using **Java** and **MySQL**. It provides a robust platform for managing quizzes, tracking user performance, and handling administrative tasks.

## Key Features

- **Quiz Management**: Create, update, and delete quizzes.
- **User Tracking**: Monitor user statistics and performance.
- **Database Integration**: efficient MySQL schema to store quiz data and user records.
- **Admin & User Portals**: Separate interfaces for administrators and quiz takers.

## Tech Stack

- **Language**: Java
- **Database**: MySQL
- **Tools**: JDBC, Swing/JavaFX (presumed for UI)
